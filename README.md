## Features

- Mượn, trả, thêm xóa sách
- Phân biệt Member và Admin
- Nhạc trong ứng dụng
- API để tìm sách
- API để tìm kiếm nhạc


## Tech Stack

- Java, JavaFX
- MySQL
- Google Books APIs
- Jamendo API
- URL các trò chơi web
